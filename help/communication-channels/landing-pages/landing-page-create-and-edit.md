# ランディングページの作成と編集 {#create-and-edit-a-landing-page}

Adobe Campaignを使用すると、オーディエンスの情報を取得し、サービスへのオファー購読、データを表示し、ランディングページを増やすために使用できるオンラインフォームのデータベースを構築できます。 ランディングページは、既存のプロファイルの取得や更新にも使用できます。
