# ランディングページの作成と編集 {#create-and-edit-a-landing-page}

Adobe Campaignを使用すると、ランディングページを作成できます。ランディングページは、オーディエンスに関する情報の取得、サービスの購読の提供、データの表示、データベースの拡張に使用できるオンラインフォームです。 ランディングページは、既存のプロファイルの取得や更新にも使用できます。
