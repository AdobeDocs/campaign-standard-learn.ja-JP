# ランディングページの作成と編集 {#create-and-edit-a-landing-page}

Adobe Campaignを使用すると、オーディエンスの情報を取得し、サービスの購読を申し込み、データを表示し、データベースを拡大するために使用できるオンラインフォームのランディングページを構築できます。 ランディングページは、既存のプロファイルの取得や更新にも使用できます。
