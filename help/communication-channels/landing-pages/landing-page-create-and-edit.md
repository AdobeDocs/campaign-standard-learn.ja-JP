# ランディングページの作成と編集 {#create-and-edit-a-landing-page}

Adobe Campaignを使用すると、オーディエンスに関する情報の取得、サービスの購読の申し込み、データの表示、データベースの拡張に使用できるオンラインフォームのランディングページを構築できます。 ランディングページは、既存のプロファイルの取得や更新にも使用できます。
